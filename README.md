# Pytorch-DDPIM

## Introduction
`Pytorch-PPDIM` is a PyTorch-based implementation of diffusion models for image generation. 

### Supported method
- [x] [DDPM](https://arxiv.org/pdf/2006.11239.pdf)
- [x] [DDIM](https://arxiv.org/pdf/2010.02502.pdf)
- [ ] []()

### Supported datasets
- [x] CIFAR-10
- [x] LSUN
- [x] CelebA
- [x] FFHQ
- [ ] ImageNet
